今回は「キーボードのＬＥＤ」を点灯させます。<br>
これでインタラプト転送とコントロール転送は分かると思います。<br>
それ以外の通信方法もだいたい同じで、送受信データをデバイスごと勉強していけば、どんなデバイスでも通信できます。<br>
まずは、必要な処理だけ作って、動けばＯＫって考え方もありですね。<br>
